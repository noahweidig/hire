## 2026-03-12 - Unified CTA button text to match section heading
 **Learning:** Inconsistencies between call-to-action buttons (e.g., 'View Work') and their destination sections/navigation links (e.g., 'Selected Work') can increase cognitive load. Unifying terminology around the destination's primary label establishes a clearer, more predictable user experience.
 **Action:** Updated the hero section CTA button from 'View Work' to 'Selected Work' to perfectly match the target section's heading and the main navigation link.
