## 2026-03-08 - Narrative Flow: Credibility Before Skills
**Learning:** Placing actual examples of work (proof/credibility) before abstract lists of skills creates a stronger, more logical narrative flow. Users see the results first, making the skills listed later more meaningful.
**Action:** When evaluating "hire me" or portfolio pages, always check if tangible proof of capability is presented before long lists of tools or skills. Reorder them if necessary to build credibility earlier in the scroll.
